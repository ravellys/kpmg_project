Documentação API - KPMG

Autor: Lucas Ravellys
Essa API trata-se de teste realizado pelo grupo KPMG. O objetivo da a API é filtrar os valores médios de carros do dataset, baseado no fabricante e na cidade. Para tanto, foi utilizada uma estrutura serveless com os componentes da AWS: Lambda, API Gateway e S3.

  1. Para o AWS Lambda, foi utilizada a seguinte função Lambda: https://kpmg-project.s3.amazonaws.com/lamda/lambda_function.py

  2. o Bucket S3 utilizado está disponível em: http://kpmg-project.s3-website-us-east-1.amazonaws.com

  3. e os endpoints criados pelo API Gateway são:

    • https://vz9x01sysl.execute-api.us-east-1.amazonaws.com/dev/valoresmedioscarros/cidades
    • https://vz9x01sysl.execute-api.us-east-1.amazonaws.com/dev/valoresmedioscarros/fabricantes

  4. Para a segurança, foi utilizada a autenticação por x-api-key
